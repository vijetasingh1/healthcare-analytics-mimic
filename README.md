# Healthcare Predictive Analytics (MIMIC-IV)

This repository contains healthcare analytics projects using ICU data from the MIMIC-IV dataset.

## Projects

### 1. Vasopressor Use Prediction
- Predicts vasopressor use using clinical features
- Includes machine learning modeling and Tableau dashboard
- Focus on treatment effects and feature interpretation

### 2. 30-Day Readmission Prediction
- Predicts patient readmission risk
- Focus on outcome prediction and risk stratification
- Uses clinical and demographic features

## Tools Used
- SQL / BigQuery ML
- Tableau
- Feature Engineering

## Note
Due to data use restrictions, MIMIC-IV dataset is not included.
